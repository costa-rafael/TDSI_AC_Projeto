# TDSI_AC_Projeto
Projeto de Arquitetura de Computadores, dedicado ao desenvolvimento em VHDL de um processador RISC simplificado de 16 bits, incluindo ALU, registos, descodificador de memória e unidade de extensão de sinal.Projeto desenvolvido em Xilinx Vivado para implementação de uma unidade de processamento em FPGA.

# Como usar
- Abrir o Vivado
- Abrir o ficheiro Unidade_Processamento_Grupo6.xpr
- Correr Synthesis, Implementation e Generate Bitstream

# Requisitos
- Xilinx Vivado instalado
- FPGA compatível
